🎂 Birthday Surprise Webpage

A beautifully animated interactive birthday surprise webpage created using HTML, CSS, and JavaScript.
This project displays a multi-step surprise message that ends with a birthday greeting and photo memory modal.

The page includes animations, gradients, confetti effects, and interactive transitions to create a memorable experience.

✨ Features
🎭 Interactive Multi-Page Flow

The website is divided into three animated pages:

1️⃣ Page 1 – Curiosity

Asks the visitor: “Do you know who you are?”

Animated question mark and gradient title

Button to continue

2️⃣ Page 2 – Reveal

Reveals a heartfelt message

Personalized text: “Pasandeeda Sakhi”

Smooth transition to the next page

3️⃣ Page 3 – Birthday Celebration

Birthday quote

"Happy Birthday Pasandeeda Sakhi"

Tags describing the person

Surprise button that opens a memory gallery

🎁 Surprise Modal

Clicking “A Surprise Awaits” opens a modal that contains:

Photo gallery

Memories section

Captions for images

Closing button

🎨 Visual Design

The page uses modern design techniques including:

Gradient animated text

Mesh animated background

Floating particles

Confetti canvas animation

Neon glow effects

Responsive layout
